# Digit Entry Views

> A digit entry view fills the entire screen and prompts people to enter a series of digits (e.g., a PIN) using a digit-specific keyboard.

**Platforms:** tvOS only

Optional title and prompt appear above the digit line.

## Best Practices

- **Use secure digit fields for sensitive data** — shows asterisks instead of entered digits.
- **Explain the purpose clearly** — title and prompt must communicate why digits are needed.
